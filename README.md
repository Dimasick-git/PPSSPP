### PPSSPP Switch — основное описание проекта


**Что это**  
Эмулятор PlayStation Portable (PSP) для консоли Nintendo Switch с полной поддержкой управления и расширенными функциями.

---

### Ключевые возможности

- **Запуск игр** в форматах ISO и CSO.  
- **Полное управление** через контроллер Switch:  
  - кнопки A/B/X/Y;  
  - аналоговые стики;  
  - триггеры (L/R, ZL/ZR).  
- **Вибрация** с поддержкой HD Rumble.  
- **Сохранения и состояния**:  
  - быстрые сохранения (savestates);  
  - стандартные сохранения PSP.  
- **Скриншоты** — сохранение кадров в момент нажатия кнопки.  
- **Кастомная иконка** приложения в меню Switch.  


---

### Структура файлов на SD‑карте


После установки проект располагается по пути:  
`/switch/ppsspp/`


**Основные папки**:
- `/PSP/GAME/` — для ISO/CSO‑файлов игр;  
- `/PSP/SAVEDATA/` — сохранения игр;  
- `/PSP/SCREENSHOT/` — скриншоты;  
- `/PSP/SAVESTATE/` — быстрые сохранения (savestates);  
- `/logs/` — логи работы эмулятора (включая `ppsspp.log`).

**Ключевые файлы**:
- `ppsspp.nro` — исполняемый файл эмулятора;  
- `ppsspp.log` — лог работы (важен для отладки).


---

### Требования

- **Nintendo Switch** (любая версия, включая OLED).  
- **SD‑карта** отформатированная в FAT32 или exFAT.  
- **Atmosphère** или другая кастомная прошивка с поддержкой Homebrew Menu.  
- **Минимум 1 ГБ свободного места** для установки и игр.


---

### Как установить


1. Скачайте архив релиза.  
2. Распакуйте содержимое в папку `/switch/ppsspp/` на SD‑карте.  
3. Поместите игры в `/switch/ppsspp/PSP/GAME/`.  
4. Запустите эмулятор через Homebrew Menu.  

---

### Как сообщить об ошибке


Если эмулятор работает нестабильно, помогите улучшить его:


**Приложите**:  
1. Файл `ppsspp.log` (из `/switch/ppsspp/logs/`).  
2. Название и регион игры (например, *Metal Gear Solid: Peace Walker (US)*).  
3. Описание проблемы:  
   - при каких действиях возникает;  
   - есть ли закономерность;  
   - скриншоты/видео (если возможно).  

**Куда отправлять**:  
В раздел **Issues** репозитория:  
https://github.com/Dimasick-git/PPSSPP


---

### Советы по использованию


- **Для лучшей производительности**:  
  - закрывайте фоновые приложения перед запуском эмулятора;  
  - используйте качественные ISO/CSO (без ошибок чтения).  
- **Резервное копирование**:  
  - регулярно сохраняйте папку `/PSP/` с сохранениями и скриншотами.  
- **Обновления**:  
  - следите за новыми релизами — многие проблемы исправляются в следующих версиях.
